1. Clone the repository
   ```sh
   git clone https://github.com/Carbon32/n8-talent
   ```
2. Install NPM packages (run inside the client folder)
   ```sh
   npm install
   ```
3. Install the required Python libraries (make a MySQL DB running in the background)
   ```sh
   pip install flask flask-jwt-extended flask-sqlalchemy flask-cors bcrypt pyms pymysql
   ```
